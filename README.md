# Rumble

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.2.14.

The idea of this project is to use agnostic vibe coding to create a simple royal rumble simulator system with queue and ring. Also with events like joining and eliminations. All changes were made using AI, though the rules were specified by developer.

The business core of the system is in the file src/app/services/rumble.service.ts

Check the agent.md for more technical info.
